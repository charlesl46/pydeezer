# pydeezer
A wrapper for Deezer's API

## Expected functionalities

- search for a deezer object
- get all infos for :
    - album
    - artist
    - chart
    - editorial
    - episode
    - genre
    - playlist
    - podcast
    - radio
    - search
    - track
    - user
- provide an api-key or oauth to allow users to modify their profiles
